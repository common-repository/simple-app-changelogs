A Plugin to display a Changelog from an App, Plugin or anything else of what you want to have a changelog off!.<br>

== Description ==

With this Plugin you can display project related changelogs on your Wordpress Site with some nice and simple animations.<br>
<br>
This could be very useful, if you want to make the user aware of each version changes and/or you have serveral project to display either on the same page or in different ones!.<br>
<br>

== Installation ==

1. Download plugin through the wordpress included plugin downloader.<br>
2. Activate plugin in plugin tab.<br>
3. Go to the plugin tab.<br>
4. Create a project, and click on View/Edit Versions.<br>
5. Create one ore more Versions!.<br>
6. Put the shortcode [appchangelog projectID=ID OF PROJECT HERE] in one of your posts or pages (To find out the id just go to the projects page, on the table there will be a value (ID)).
<br>
7. Now you have a changelog on the page :D!

Sure you can, these are located in the assets folder from the plugin, hoever make a copy of you custom code, because for now custom css and js will get overwritten after each update D: (Will add an fature in the future to do custom css without the need to worry after each update)!
